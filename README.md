##  Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or text editor
- Basic understanding of Java programming

### How to Run

1. **Clone or download this repository**
   ```bash
   git clone <repository-url>
   cd Data_Structures_And_Algorithm
   ```

2. **Navigate to a specific week folder**
   ```bash
   cd Week1_Introduction_to_DSA
   ```

3. **Compile a Java file**
   ```bash
   javac Hello_DSA.java
   ```

4. **Run the compiled program**
   ```bash
   java Hello_DSA
   ```

### Example Usage

```bash
# Navigate to Week 1
cd Week1_Introduction_to_DSA

# Compile and run
javac Hello_DSA.java
java Hello_DSA
```
